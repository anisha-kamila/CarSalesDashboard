# Car Sales Dashboard

## Project Overview
Developed an interactive **Car Sales Dashboard** in **Power BI** to analyze and monitor key business metrics such as sales performance, average car price, cars sold, dealer performance, and regional sales trends. The dashboard enables data-driven decision-making through dynamic visualizations and filters.

---

## Dashboard Features

### Overview Dashboard
- YTD (Year-to-Date) Total Sales, Average Price, and Cars Sold KPIs
- MTD (Month-to-Date) performance comparison
- Weekly Sales Trend analysis
- Sales breakdown by **Body Style**
- Sales distribution by **Car Color**
- Dealer Region-wise sales using **Map Visualization**
- Company-wise sales trend and performance comparison

### Details Dashboard
- Transaction-level sales details
- Customer-wise and dealer-wise sales records
- Vehicle model and company performance tracking
- Interactive filtering by:
  - Body Style
  - Dealer Name
  - Transmission
  - Engine
